# RAGChat
RAG问答系统  

运行方式： 
1. (cmd)激活虚拟环境  
venv\Scripts\activate  

2. 构建知识图谱  
python index.py --root ./ragtest

3. 执行查询
python query.py --root ./ragtest --method local "What are the top themes in this story?"  
python query.py --root ./ragtest --method local "Who is Bob Cratchit"
